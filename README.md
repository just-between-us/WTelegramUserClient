## О проекте

Важное отличие от обычных ботов:
Обычные Telegram боты работают через Bot API и имеют ограниченный доступ.
Этот проект использует библиотеку WTelegram.Client для прямого подключения к MTProto протоколу Telegram, что позволяет авторизоваться как обычный пользователь.

Это открывает возможности, недоступные для ботов:

- Чтение сообщений из любых личных чатов
- Доступ к контактам и истории
- Реакция на статус "онлайн/офлайн" собеседников
- Полный контроль над аккаунтом

Проект написан на C# 12 под .NET 9, имеет модульную архитектуру, гибкую систему конфигурации и удобный консольный интерфейс.

## Возможности

Категория Функция
- 🔐 Авторизация Вход по номеру телефона, работа через MTProto
- 👂 Мониторинг Реальное время: прослушивание входящих сообщений и событий
- ✍️ Отправка Отправка текстовых сообщений, поддержка ответов
- ⚙️ Конфигурация Сервис для хранения API ID, Hash, номера телефона
- 🖥️ Интерфейс - консольное управление
- 📦 Современный стек .NET 9, C# 12, асинхронное программиование

## 🚀 Быстрый старт

Предварительные требования

· Установленный .NET 9 SDK
· Аккаунт Telegram
· API ID и API Hash (получить на my.telegram.org)

Установка и запуск

1. Клонируйте репозиторий
   ```bash
   git clone https://github.com/just-between-us/WTelegramUserClient.git
   cd WTelegramUserClient
   ```
2. Запустите
   ```bash
   dotnet run
   ```
3. Следуйте инструкциям в консоли
   - Задайте API ID и API Hash
   - Укажите номер телефона 
   - Введите код из Telegram
#### Готово! Клиент запущен и слушает сообщения 🎉
